一些常用的命令

下列一些常用的uv命令

  • uv init: 创建新 Python 项目
  • uv add: 为项目添加依赖
  • uv remove: 从项目移除依赖
  • uv sync: 同步项目依赖到环境
  • uv lock: 为项目依赖创建锁文件
  • uv run: 在项目环境中运行命令
  • uv tree: 查看项目依赖树
  • uv build: 构建项目为分发包
  • uv publish: 发布项目到包索引
