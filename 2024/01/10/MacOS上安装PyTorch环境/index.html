MacOS上安装PyTorch环境

2024-01-10 | 人工智能

本地化安装 PyTorch 环境需要以下几个步骤

  1. 安装Anaconda
  2. 创建并激活虚拟环境
  3. 安装 PyTorch 库
  4. 安装 TensorBoard 库
  5. 安装 Graphviz 软件和 TorchViz 库（可选）
  6. 安装 git 并clone 资源
  7. 启动 Jupyter notebook

接下来我们一步步的完成上述的安装过程

安装Anaconda

Anaconda 的下载地址为

  • Windows (https://docs.anaconda.com/anaconda/install/windows/)
  • macOS (https://docs.anaconda.com/anaconda/install/mac-os/)
  • Linux (https://docs.anaconda.com/anaconda/install/linux/)

我们选择安装 macOS 版本的，然后根据安装向导一步步安装，记住一定要安装 Python3.0 版本的。

安装完成后就可以创建一个虚拟环境了。

创建Conda 虚拟环境

虚拟环境是一个隔离 Python 安装环境的好方法，通过这种方式你可以在电脑上安装不同版本依赖的 Python 环境，而不会出现不同Python 依赖的冲突。我应该总是为每个项目创建一个单独的虚拟环境，避免被其他项目的依赖“污染”。
