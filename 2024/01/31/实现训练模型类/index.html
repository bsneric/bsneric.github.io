定义构造函数

Python 中的构造函数是__init__(self)，通常它用来定义初始化加载的属性参数。一般包括：

  • 参数：由调用的用户提供
  • 占位符：在初始化类时无法马上提供的参数
  • 变量：我们要全局操作的变量
  • 函数：通常是利用高阶函数功能定义的可以实现动态参数的函数。

Methods，_Methods和__Methods

类似 Java 这样的语言有三种方法类型：public，protected 和 private。

  • protected: 是受保护的方法，它只能在类的内部以及类的字类调用。
  • private：是私有方法，只能在类的内部调用
  • public：是公开方法，可以在类的内外部调用

这些规则在 Java 中是适用的，但是 Python 不行。Python 中所有的方法都是公开的，就是说你可以随意调用。但是有一个约定的建议，那就是对于受保护的方法，一般用_ 一个下划线开头，对于私有方法用__ 两个下划线开头，它实际没有对方法可访问性的限制，只是对方法调用者的一种提示。

我们刚才定义的_make_train_step_fn和_make_val_step_fn就是受保护的方法，它告诉调用者这两个方法是受保护的，如果有人继承了 StepByStep 这个类，那么这两个方法是可以调用的。

刚才那两个方法可以直接写在类中，但是后续我们还要增加其他方法，为了不每次都修改我们的类，下面用了一种 Python 中的特有方式将这两个方法“注入”到类中。
